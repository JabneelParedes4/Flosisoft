Crud de Ventas

Se creó el CRUD para el subsistema de ventas, y en él se incluye el formulario que se utilizó para registrar a los clientes.

Crud de Ventas

Se muestran los registros que existen, los cuales serán utilizados para observar su funcionamiento, donde se aplica:

  • Crear
  • Leer
  • Actualizar
  • Eliminar

Crear Cliente

Registrar un cliente implica completar varios campos, tales como:

  • Nombre
  • Correo
  • Teléfono
  • Dirección
  • RFC
  • Régimen Fiscal

Actualizar Cliente

Actualizar la información de un cliente permite tener datos más recientes. Por ejemplo, se puede cambiar el correo electrónico o teléfono, etc.

Eliminar Cliente

Eliminar un cliente ayuda a mantener nuestro sistema de registros de la florería más ordenado.

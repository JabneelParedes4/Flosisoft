Subsistema de Ventas

El subsistema de ventas se encarga de registrar todas las transacciones. Gestiona la facturación, los métodos de pago y genera informes sobre el rendimiento de las ventas, lo que es fundamental para evaluar la rentabilidad del negocio.

Usuarios del subsistema de Proveedores

  • Cliente
  • Dueña
  • Empleado de ventas

Requerimientos por usuario

El cliente debe ser capaz de:

  • Acceso a la Información
    • Permitir al usuario acceder y administrar sus datos.
  • Visualización del Catálogo
    • Visualizar el catálogo de productos.
    • Filtrar el catálogo por:
      • Precio
      • Tema
      • Popularidad
  • Diseño de Ramo Personalizado
    • Elegir tipo de flor.
    • Especificar cantidad de flores.
    • Incluir decoraciones extra.
    • Ordenar el Ramo
  • Ingresar información de entrega:
    • Dirección de entrega
    • Referencia
    • De y Para
    • Teléfono
    • Fecha de entrega
    • Hora
  • Proceso de Pago
    • Tarjeta:
      • Número de tarjeta
      • Cuenta
      • Fecha de vencimiento
      • CVV
    • Transferencia:
      • Insertar comprobante de pago

La dueña debe ser capaz de:

  • Supervisión y Gestión
    • Visualizar la lista de pedidos actuales.
    • Consultar historial de compras.
  • Estado del Pedido
    • Pendiente
    • En Proceso
    • Entregado
    • Cancelado
  • Detalles del Pedido
    • Fecha de pedido (cuando se realizó).
    • Fecha de entrega (programada).
    • Monto total del pedido.
  • Consultar detalles específicos de cada pedido:
    • Ver el contenido del ramo.
    • Datos del cliente.
  • Coordinación de Entregas
    • Organizar y programar entregas.

El empleado de ventas debe ser capaz de::

  • Gestión de Pedidos y Atención al Cliente
    • Revisar información del cliente:
      • Nombre
      • Dirección de entrega
      • Teléfono
      • Fecha y hora de entrega
    • Confirmación de Detalles de Pedidos
      • Tipo de flores
      • Cantidad
      • Personalización
    • Atención a Consultas
      • Responder preguntas.
      • Informar precios y promociones.
    • Procesar Pagos
      • Métodos de pago:
        • Tarjeta
        • Transferencia
        • Efectivo
